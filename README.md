# Russia-whitelist-rules-gen

[RU](/README.md) | [EN](/README-en.md)

Проект содержит список российских интернет-сервисов, которые остаются доступными в условиях ограничения интернета в России, и Python скрипт для генерации набора правил для VPN клиентов, основанных на xray-core или sing-box.

## Предыстория

В России вводятся "белые списки", когда доступен только интернет-сервисы из whitelist. Этот список был взят с [сайта Мегафона](https://moscow.megafon.ru/offline_services/#whitelist).

## Структура проекта

- [`whitelist.yaml`](whitelist.yaml) - YAML-файл с категоризированными записями whitelist
- [`gen_list.py`](gen_list.py) - Python-скрипт для генерации правил маршрутизации из whitelist

## Категории

Whitelist включает сервисы следующих категорий:

- **Телеком** - Мобильные операторы (МегаФон, МТС, Билайн, Tele2, Yota)
- **Госсервисы** - Государственные услуги, органы власти и госкомпании (Госуслуги, Госключ, Центробанк и др.)
- **Финансы** - Банки и платёжные системы (Мир, СБП, Московская биржа, Альфа-Банк, ВТБ, ПСБ)
- **Ритейл и еда** - Интернет-магазины и доставка еды (Ozon, Wildberries, Самокат, Пятёрочка, Магнит и др.)
- **Медиа и IT** - Телеканалы, стриминговые сервисы, новостные порталы и IT-компании (Яндекс, VK, Кинопоиск, Rutube, ИВИ и др.)
- **Транспорт** - Транспорт и путешествия (РЖД, Аэрофлот, Туту и др.)
- **Другое** - Различные сервисы (2ГИС, Авито, HeadHunter, СДЭК и др.)

## Установка зависимостей

Перед запуском скрипта установите необходимые зависимости:

```bash
pip install pyyaml
```

## Использование

Запустите `gen_list.py` для генерации правил маршрутизации:

```bash
# Сгенерировать правила для xray-core клиентов
python gen_list.py x

# Сгенерировать правила для sing-box клиентов
python gen_list.py s

# Сгенерировать оба формата
python gen_list.py
```

## Форматы вывода

- **xray-core**: Использует префиксы `domain:` и `geosite:`
- **sing-box**: Преобразует `geosite:` в префикс `set:geosite-`
